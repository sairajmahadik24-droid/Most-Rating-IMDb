#  Movie Industry Analysis 

Project Overview

This project explores movie industry data using SQL to analyze ratings, revenue performance, director contributions, and movie trends over time.
The dataset contains detailed information about movies including title, genre, director, revenue, runtime, and ratings.
The objective is to extract insights about movie performance and industry trends.


# Business Objectives

* Identify highest rated movies
* Analyze movies generating highest revenue
* Evaluate director performance**
* Study movie release trends by year
* Understand relationship between ratings and revenue

# Dataset Description

The dataset includes movie information such as:

* Movie title
* Genre
* Director
* Actors
* Release year
* Runtime
* Rating
* Votes
* Revenue
* Metascore

# SQL Analysis Performed

Key analysis performed in this project:

* Total number of movies
* Average movie rating
* Top 10 highest rated movies
* Top 10 highest revenue movies
* Movies released per year
* Average rating by year
* Directors with most movies
* Directors with highest average rating
* Most popular movies by votes
* Movies with rating above 8
* Best movie each year using window functions


# SQL Concepts Used

* Aggregate Functions
* GROUP BY
* ORDER BY
* Subqueries
* Window Functions (RANK)
* Data Aggregation

# Key Insights

* Certain directors consistently produce highly rated movies.
* High ratings do not always guarantee high revenue.
* Movie production trends vary across different years.
* Some movies achieve exceptional popularity based on audience votes.
